# Clojure
## Переменные
Чтобы определить переменную нужна специальная форма `def`. Форма, потому что первый аргумент нельзя вычислить.
Пример:
```
(def x 10)
```
Однако переменные в Clojure менять нельзя.

## Преобразования
```
(<type> <expression>)
```

## Поверка на тип
```
(<type>? <expression>) // -> boolean
```

## Вывод
```
(println [аргументы])
```

## Функции
```
(defn <func-name> [arguements] (
    <body>
)
```
